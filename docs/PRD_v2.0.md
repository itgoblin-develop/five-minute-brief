## 1. Executive Summary

본 문서는 **출근길 모바일 뉴스레터 서비스 '오늘 5분'**의 MVP 개발을 위한 통합 제품 요구사항을 정의합니다. 기획팀, 디자인팀, 개발팀의 문서를 통합하고 실제 구현 현황을 반영하여 최신화하였습니다.

### 핵심 가치 제안

바쁜 직장인들이 **출근 시간 5분 이내에 오늘의 주요 뉴스를 카드 형태로 빠르게 소비**할 수 있도록 설계된 모바일 우선 웹 서비스입니다.

**(가칭) 오늘 5분 : 주 5일 출근하는 날, 늘 5분간 볼 수 있는 출근길 모바일 뉴스레터 서비스**

### 주요 특징

- 선택의 피로 없이 큐레이션된 핵심 뉴스만 제공
- 카드 뉴스 형태의 직관적인 스와이프 UI + 리스트 뷰 전환
- 카테고리별 필터링 (전체, 트렌딩, 경제, 재테크, 사회)
- 사용자의 뉴스 열람 패턴을 학습하여 최적의 시간에 푸시 알림 발송
- 비로그인 사용자는 3개 레터 확인 가능, 나머지는 썸네일로만 노출
- AI 재구성 뉴스 콘텐츠로 가독성 높은 뉴스 소비 경험 제공
- 좋아요/북마크/댓글 소셜 인터랙션 기능

## 2. 배경 및 목적

### 2.1 배경 (Background)

현대 직장인들이 겪는 뉴스 소비의 문제점:

1. **선택의 피로**: 아침 출근 시간에 어떤 뉴스를 읽어야 할지 결정하기 어려움
2. **시간 부족**: 긴 기사를 읽을 시간이 없지만, 트렌드와 필수 이슈는 놓치고 싶지 않음
3. **정보 과부하**: 기존 뉴스 앱은 너무 많은 정보를 제공해 핵심만 빠르게 파악하기 어려움
4. **맞춤형 부재**: 개인화된 뉴스 큐레이션 서비스는 있지만, 출퇴근 시간대에 특화된 서비스는 부족

### 2.2 목적 (Objectives)

1. 출근 시간 10분 이내에 소비 가능한 요약 뉴스를 카드 형태로 제공하는 모바일 뉴스레터 MVP 개발
2. 카테고리별 뉴스 필터링 기능 구현
3. 사용자의 뉴스 열람 패턴을 분석하여 최적의 시간에 푸시 알림 발송 (알림 허용 시)
4. 프로덕션 배포까지 완료하여 실사용자 피드백 수집

### 2.3 성공 기준 (Success Criteria)

**기능 기준**

- 핵심 플로우 4개(메인 피드, 카테고리 탭, 로그인, 상세보기)가 오류 없이 동작
- 뉴스 API 연동이 정상 작동하여 실시간 뉴스 제공

**품질 기준**

- 페이지 로딩 시간 p95 **3초 이내**
- 푸시 알림 발송 성공률 **95% 이상**
- 모바일 반응형 지원 (iOS 14+, Android 10+)

**산출 기준**

- 프로덕션 배포 URL 제공
- 사용자 가이드 1페이지 작성
- API 문서화 완료

**사용자 지표 (출시 후 2주)**

- DAU 100명 이상
- 평균 세션 시간 5분 이상
- 뉴스 카드 평균 3개 이상 읽기

## 5. 핵심 기능 (Core Features)

### 5.1 MVP 포함 기능 ✅

### 1. 메인 뉴스 피드

- 카드 형태의 스와이프 UI로 뉴스 제공
- **[v2.0 추가]** 리스트 뷰 전환 토글 (카드 뷰 ↔ 리스트 뷰)
- 제목, 2~3줄 요약, 이미지, 날짜 표시
- 좌우 스와이프로 다음 뉴스 이동 (카드 뷰)
- 무한 스크롤 (리스트 뷰, 하단 100px 도달 시 추가 로드)

### 2. 카테고리별 필터링

- 5개 탭: 전체, 트렌딩, 경제, 재테크, 사회
- 테크/IT 뉴스는 트렌딩 카테고리에 포함
- 수평 스크롤, 필(pill) 형태 버튼

### 3. 요약 뉴스 제공

- 제목 + 2~3줄 요약 (200자 이내)
- 해시태그 표시 (뉴스 키워드 기반)
- 썸네일 이미지 (AI 재구성)
- 좋아요/북마크/댓글 수 표시

### 4. 재구성된 뉴스 콘텐츠

- 원문을 읽기 쉽게 재구성한 콘텐츠
- 스크롤 프로그레스 인디케이터
- **[v2.0 추가]** 공유 기능 (카카오톡, 링크 복사)
- **[v2.0 추가]** 댓글 섹션 (댓글 입력, 수정, 삭제)

### 5. 로그인/회원가입

- 이메일 기반 인증
- **[v2.0 추가]** 이메일 인증 코드 방식 (회원가입 시)
- **[v2.0 추가]** 비밀번호 표시/숨김 토글
- **[v2.0 추가]** 약관 동의 (만 14세 이상, 서비스 이용약관, 개인정보처리방침)
- JWT 토큰 기반 세션 관리 (90일 유효)

### 6. 비로그인 제한

- 3개 레터까지 전체 내용 확인 가능
- 4번째 레터부터 썸네일만 노출
- 썸네일 클릭 시 로그인 유도 팝업

### 7. 푸시 알림 (스마트 타이밍)

- 알림 허용 시에만 작동

## 16. 데이터 정책 및 법적 리스크

### 수집 방식

- 뉴스 API (네이버 뉴스, 유튜브)
- 사용자 입력 (회원 정보)

### 저장 데이터

- 뉴스 원문: 저장하지 않음
- 뉴스 메타데이터: 제목, 요약, 날짜
- 사용자 정보: 이메일(암호화), 해싱된 비밀번호, 디바이스 토큰

### 보관 기간

- 사용자 데이터: 회원 탈퇴 시 즉시 삭제
- 로그 데이터: 30일
- 뉴스 데이터: 7일

### 금지 데이터

- 주민번호, 계좌번호, 카드번호 등 민감정보
- 뉴스 원문 전체 복사 (저작권 침해 방지)
- 사용자 행동 추적 데이터 (위치, 브라우징 기록 등)

### 개인정보 처리방침 필수 항목

1. 수집하는 개인정보 항목 (이메일, 닉네임, 디바이스 토큰)
2. 개인정보 수집 및 이용 목적 (서비스 제공, 푸시 알림)
3. 개인정보 보유 및 이용 기간 (회원 탈퇴 시까지)
4. 개인정보 제3자 제공 여부 (제공하지 않음)
5. 개인정보 처리 위탁 여부 (FCM 서버)
6. 이용자 권리 및 행사 방법 (열람, 정정, 삭제, 탈퇴)
